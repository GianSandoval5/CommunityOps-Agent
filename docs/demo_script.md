# Demo Script

## 0:00 - Problem

Community events are usually coordinated through chats, spreadsheets, forms, and memory. This creates missed tasks, weak follow-up, and repeated planning work.

## 0:25 - Solution

CommunityOps Agent is an AI operations assistant for tech communities. It uses Gemini and Google Cloud Agent Builder for reasoning and MongoDB MCP as event memory and action layer.

## 0:50 - Main Prompt

```txt
Organize a Flutter Piura meetup for 80 people about Flutter + AI.
Budget: S/500.
Needs: 2 talks, coffee break, and certificates.
```

## 1:10 - Agent Planning

Show that the agent reads community memory, recommends speakers, checks sponsors, creates an agenda, detects risks, and drafts messages.

## 1:45 - Human Approval

Show the approval button. Explain that the agent does not write important data without organizer confirmation.

## 2:05 - Execution

After approval, show the dashboard updated:

- Event readiness score.
- Agent run timeline.
- Tasks.
- Risks.
- Message drafts.

## 2:45 - Close

CommunityOps Agent helps communities run better events with less manual work, persistent memory, and supervised automation.
